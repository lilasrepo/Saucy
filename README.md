# Saucy（繁中移植版 · TC12） / Traditional-Chinese Port

> 當你懶得自己刷 MGP 的時候。<br>
> When working for MGP is too much effort.

**繁體中文**：這是 **[Saucy](https://github.com/PunishXIV/Saucy)** 的繁體中文客戶端移植版，對應 **FFXIV 7.1 / yanmucorp Dalamud API12（.NET 9）**。本專案僅做相容性移植，**非官方、非原作維護**；所有原始功能與設計著作權歸原作者 **Taurenkey、Kagekazu**。

**English**: A Traditional-Chinese-client port of **[Saucy](https://github.com/PunishXIV/Saucy)** targeting **FFXIV 7.1 / yanmucorp Dalamud API12 (.NET 9)**. Compatibility port only — **unofficial and not maintained by the original author**. All original work © **Taurenkey, Kagekazu**.

## 這是什麼 / About

自動進行部分金碟遊樂場（Gold Saucer）小遊戲，幫你輕鬆累積 MGP。

Automates certain Gold Saucer mini-games to farm MGP for you.

## 安裝 / Installation

**繁體中文**
1. 使用 **XIVTCLauncher** 啟動繁體中文客戶端。
2. 遊戲內輸入 `/xlsettings` → 切到 **Experimental** 分頁 → **Custom Plugin Repositories（自訂插件庫）**。
3. 貼上下列網址並按 **+** 儲存：
   ```
   https://raw.githubusercontent.com/lilasrepo/DalamudPlugins/main/pluginmaster.json
   ```
4. 輸入 `/xlplugins`，搜尋 **Saucy (TC12)** → 安裝 → 啟用。

**English**
1. Launch the Traditional-Chinese client with **XIVTCLauncher**.
2. In-game, type `/xlsettings` → **Experimental** tab → **Custom Plugin Repositories**.
3. Add this URL and save with **+**:
   ```
   https://raw.githubusercontent.com/lilasrepo/DalamudPlugins/main/pluginmaster.json
   ```
4. Type `/xlplugins`, search **Saucy (TC12)** → Install → Enable.
